>#### Esse é um simples aplicativo de cálculo de juros compostos, o usuário insere
>#### alguns dados a cerca do investimento feito e o programa realiza o cálculo e exibe
>#### o resultado para o usuário. Esse programa tem como intuito reforçar operações
>#### aritiméticas, o uso de variáveis com valores adicionados pelo usuário e o uso da
>#### classe Math.
